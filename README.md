# Intern Performance Prediction Model

## Objective

To build a machine learning model that predicts intern performance based on task completion time, feedback ratings, and attendance.

## Model Used

* Random Forest Regression

## Results

* R² Score: 0.94
* MSE: 4.60

## Features

* Task Completion Time
* Feedback Ratings
* Attendance

## Output

* Predicts performance score
* Classifies interns as:

  * High Performer
  * Average Performer
  * Needs Improvement

## Visualizations

* Feature Importance Graph
* Actual vs Predicted Graph

## Conclusion

The model successfully predicts intern performance and helps identify high-performing and struggling interns.
